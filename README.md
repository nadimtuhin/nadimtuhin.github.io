# nadimtuhin.github.io

Personal website and blog for Nadim Tuhin, hosted at [nadimtuhin.com](http://nadimtuhin.com). Built with Hugo using the Ghostwriter theme.

## What it is

A static site generated by Hugo 0.20.7, containing a personal blog with posts, an RSS feed, and a sitemap. The site is served via GitHub Pages with a custom domain configured through a CNAME record.

## Tech stack

- **Generator**: Hugo 0.20.7
- **Theme**: Ghostwriter (ported to Hugo by jbub)
- **Styles**: SCSS compiled to CSS, served from `dist/styles.css`
- **JavaScript**: jQuery 1.11.3, highlight.js, jquery.fitvids

## Structure

```
nadimtuhin.github.io/
├── index.html          # Generated homepage
├── post/               # Blog post pages
├── tags/               # Tag index pages
├── categories/         # Category index pages
├── styles/             # SCSS source files
│   ├── style.scss
│   └── highlight.css
├── js/                 # JavaScript files
├── dist/               # Compiled CSS output
├── sitemap.xml
├── index.xml           # RSS feed
└── CNAME               # Custom domain: nadimtuhin.com
```

## Local development

Install Hugo (0.20.x recommended for compatibility):

```bash
brew install hugo
# or download from https://github.com/gohugoio/hugo/releases/tag/v0.20.7
```

Run the development server:

```bash
hugo server -D
```

Open `http://localhost:1313` in your browser.

## Building

```bash
hugo
```

Hugo outputs the static site to the `public/` directory.

## Deployment

The site deploys automatically via GitHub Pages. Push to the default branch and GitHub serves the content. The `CNAME` file routes traffic from `nadimtuhin.com` to the GitHub Pages URL.

## Adding a post

```bash
hugo new post/my-post-title.md
```

Edit the generated file in `content/post/`. Set `draft: false` when ready to publish.
